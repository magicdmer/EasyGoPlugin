# EasyGo 插件仓库

## 工程结构

**depends**:  EasyGo一些插件依赖的第三方软件，比如解码器啥的，支持音乐播放

**plugin**:  插件目录，格式为 `.plugin`  ，其实就是 zip 格式改了后缀，具体可以看压缩包自带的文档

**plugin-sources**:  这个里面是插件源码，主要是c++和易语言的，因为python本身就是源码，可以解压看



## 注意事项
你可以fork或者参考该工程结构，弄一个自己的插件仓库，供自己使用
